# ゼロつく2 第一章

## 学び
- ベクトルや行列(N=2) を拡張させて N 次元の集まりのことを**テンソル**と呼ぶ.
- 「要素ごとの」とは英語で element-wise と言う. NumPy の多次元配列に対して四則演算などできる.
- 一方で, 形状の異なる配列同士も演算が可能で**ブロードキャスト**という.
- **100 numpy excercises** って言う NumPy の練習問題がある.
- ニューラルネットワークで**スコア**と言う言葉が出てきたらそれは「確率」になる前の値で, Softmax 関数に入れることで, 確率が得られる.
- ニューラルネットにおいて**レイヤ**とは最小単位の処理を表し, レイヤが集まってネットワークを構成する. 例えば, Affine レイヤは線形変換を行うレイヤで, ReLU レイヤは非線形変換を行うレイヤ. Affine レイヤと ReLU レイヤを組み合わせることで, より複雑なネットワークを構築できる. Sigmoid レイヤは出力を0から1の間に変換するレイヤで, 出力が確率を表す場合などに使用される.
- レイヤは `forward()`, `backward()`と言うメソッドをもち, インスタンス変数として`params`と`grads`を持つ.
- 多クラス分類を行う場合は損失関数として**交差エントロピー誤差**が利用される. ただしこれは教師ラベルがある場合.
## コードについて
- 'np.array()' メソッドでベクトルや行列を生成. これによって NumPy の多次元配列用のクラスである, `np.ndarray` クラスが生成される.
- `np.dot(A, B)` で行列 A と B のドット積 (行列の積) を計算.
